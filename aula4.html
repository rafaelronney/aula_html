Aula 4 - div's, span's e elementos block (bloco) e inline

elementos html são compostos de tags de abertura e fechamento (como, por exemplo um link ou um parágrafo), ou apenas uma tag que possui uma / no final (caso do elemento img).

Por padrão, se nada for modificado via css, um div, assim como todo elemento block (bloco) ocupará toda largura disponível para o mesmo. O mesmo não acontece com objetos inline como esse link (elemento <a>) ou esse span (elemento).

São exemplos de elementos block:

  1. <div>
  2. <p>
  3. <ul> (lista simples)
  4. <ol> (lista numerada, como essa)
  5. <table>

São exemplos de elementos inline:

  • <a>
  • <img>
  • <span>
  • <input>
  • <select>
  • <label>
  • <abbr>
  • <textarea>
Mas, é importante saber que elementos inline podem ser transformados em bloco, como esse <span> que você está vendo aqui.
